Project Overview

PowerCast forecasts the Market Clearing Price (MCP) on India's IEX power exchange using machine learning on 15-minute historical block data.

  • Target: MCP (₹/MWh)
  • Frequency: 96 blocks/day (15-min)
  • Train/test split at 2025-01-01
  • Best model: Random Forest (~9.09% MAPE)
🔬

Feature Engineering

  • MCP_lag_1, MCP_lag_2 — immediate lags
  • MCP_lag_96 — same time yesterday
  • MCP_lag_672 — same time last week
  • MCP_roll_mean_96 — 24h rolling mean
  • MCP_roll_std_96 — 24h rolling std
  • hour, weekday, is_weekend, is_peak_hour
🛡

Anti-Leakage Strategy

Four columns removed before training:

  • MCP_capped — derived from target
  • price_lag_1 — duplicate of MCP_lag_1
  • price_lag_2 — duplicate of MCP_lag_2
  • price_roll_mean_4 — derived from target
⚡

Performance Optimisations

  • Model trained once at startup (warm_up)
  • Pre-allocated NumPy array for predictions
  • deque(maxlen=96) rolling buffer
  • Direct numpy.reshape(1,-1) predict calls
  • Hybrid forecast for large horizons
  • CSV dataset cache (10–50× faster loading)
🔄

Forecast Horizons

  • Daily: 96 blocks · full model
  • Weekly: 672 blocks · hybrid
  • Monthly: 2,880 blocks · hybrid
  • Seasonal: 8,640 blocks · hybrid
  • Custom date: auto-computes blocks · hybrid if large
